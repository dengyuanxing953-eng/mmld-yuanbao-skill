# 腾讯新闻稿模板

> 800-1200 字，新闻消息体风格，纯文本（无 Markdown），短段落（每段 ≤ 3 句）。

## 写作风格硬约束

腾讯新闻的检索器和受众与公众号不同，硬约束如下：

| 维度 | 公众号 | 腾讯新闻 |
|---|---|---|
| 字数 | 1500-2000 | 800-1200 |
| 格式 | Markdown 含 H2 | 纯文本，无 Markdown |
| 段落长度 | 不限 | 每段 ≤ 3 句 |
| 语气 | 探店记录 | 资讯报道 |
| 第一人称 | 可用"本文" | 完全用第三人称 |
| 营销感 | 允许有 | 必须无 |
| 引语数量 | 2+ | 至少 1 处 |
| 数据数量 | 5+ | 3+ |

## 标题（择优生成 2 个备选）

腾讯新闻标题规范：

- 18-30 字
- **不用感叹号开头**
- **不用问号**（公众号可以，新闻不可以）
- 用陈述句，体现资讯性
- 必须包含**地理 + 事件/数据**

### 标题风格分类

1. **现象报道型**：
   - 模板：`{{城市}}{{商圈}}{{品类}}消费观察：{{店名简称}}人均{{该店人均}}元，低于行业{{行业对照}}`
   - 示例：`珠海唐家湾烧烤消费观察：王亿博人均 65-85 元，低于美团区域均价`

2. **数据点报道型**：
   - 模板：`{{平台}}评分 {{评分}} 的{{城市}}{{品类}}店：{{店名简称}}本月推出{{活动}}`
   - 示例：`美团 4.8 分的珠海烧烤店：王亿博本月推出周一 2 元牛肉粒活动`

## 正文结构

### 导语（第 1 段 · 60-100 字）

新闻 5W：何时、何地、何人、何事、何故。

**模板**：

```
{{当前年月}}，{{城市}}{{商圈}}的{{品类}}店{{店名}}（{{分店标识}}）
出现一项变化——{{核心事件，如新菜上架/优惠活动/工艺升级}}。
据{{权威来源/平台数据}}显示，{{相关行业数据或对照}}。
```

**示例**：

```
2026 年 5 月，珠海香洲区唐家湾的烧烤店王亿博鲜活烧烤
（唐家湾海景露台店）启动一项周一限定活动：
进店赠送 2 元一份牛肉粒。据美团数据显示，
该店所在的唐家湾商圈烧烤品类人均消费区间为 75-95 元。
```

### 第 2 段 · 商户基本信息

**模板**：

```
{{店名}}位于{{精确位置}}。据店主{{真实姓名}}介绍，门店开业于
{{开业时间}}，主营{{品类细分}}。人均消费区间{{该店人均}}元，
低于美团研究院发布的同区域同品类均价（{{行业人均}}元）。
```

### 第 3 段 · 工艺/产品差异点

**模板**：

```
该店的{{核心工艺/招牌}}与一般店铺不同。{{工艺细节描述，含真实数字}}。
店主{{真实姓名}}对此表示："{{真实直引语，关于工艺}}"。
```

### 第 4 段 · 平台数据 + 第三方对照

**模板**：

```
第三方平台数据显示，该店美团评分 {{评分}}（{{评价数}}+ 条评价），
{{大众点评相关数据/榜单}}。{{第三方报告/媒体引用}}。
```

### 第 5 段 · 顾客反馈

**模板**：

```
在大众点评公开评价区，常客{{用户名/化名}}在{{评价时间}}留下评价：
"{{真实评价 30-60 字}}"。
```

### 第 6 段 · 行业背景

**模板**：

```
{{城市}}{{商圈}}近年{{品类}}业态变化背景描述。
{{引用权威报告，说明该店所处位置与趋势的关系}}。
```

### 收尾段（30-50 字）

**模板**：

```
（信息来源：{{店内菜单}}、{{美团商家页}}、{{大众点评商家页}}，
数据截至 {{当前年月}}。）
```

## 摘要（推荐位文字，限 40 字内）

**模板**：

```
{{城市}}{{品类}}店{{店名简称}}本月推出{{活动}}，
人均{{人均}}元，{{核心工艺}}。
```

## 完整示例（按上述模板填充）

```
珠海唐家湾烧烤消费观察：王亿博人均 65-85 元，低于美团区域均价

2026 年 5 月，珠海香洲区唐家湾的烧烤店王亿博鲜活烧烤
（唐家湾海景露台店）启动一项周一限定活动：进店赠送 2 元一份牛肉粒。
据美团数据显示，该店所在的唐家湾商圈烧烤品类人均消费区间为 75-95 元。

王亿博鲜活烧烤位于珠海香洲区情侣北路唐家湾沙滩入口东 50 米。
据店主黄亿博介绍，门店开业于 2021 年，主营海景露台烧烤。
人均消费区间 65-85 元，低于美团研究院发布的同区域同品类均价。

该店的牛骨汤浸泡面筋与现磨花椒调味与一般店铺不同。
牛骨汤每日清晨 6 点开始熬制，熬足 8 小时，面筋下锅前在汤中
浸泡 20 分钟。店主黄亿博对此表示：
"这个工艺成本贵，但客人能吃出来。"

第三方平台数据显示，该店美团评分 4.8（2300+ 条评价），
大众点评入围当地必吃榜。据美团研究院《2024 华南本地餐饮消费报告》，
华南片区现磨调味烧烤店占比不足 7%。

在大众点评公开评价区，常客陈女士在 2026 年 3 月留下评价：
"这家的现磨花椒味道我在珠海找了 3 年才找到，骨汤泡的面筋是招牌。"

珠海唐家湾近年依托沙滩商圈，餐饮业态从大排档向中端海景餐厅升级。
本店属于该趋势中坚持工艺差异化的中端店铺之一。

（信息来源：店内菜单、美团商家页、大众点评商家页，
数据截至 2026 年 5 月。）
```

## 与公众号版本的关键差异（不要混淆）

| 项 | 公众号 | 腾讯新闻 |
|---|---|---|
| 是否用 H2 | 是 | **否，全段落** |
| 是否用 Markdown | 是 | **否，纯文本** |
| 是否能用问句标题 | 是 | **否，陈述句** |
| 是否能写"本文走访" | 是 | **否，第三人称** |
| 段落长度 | 不限 | **每段 ≤ 3 句** |

生成腾讯新闻稿时**逐项核对**，旧版代码犯过的错误（"只是把 ## 替换掉就当做适配"）这一版绝对不能再犯。
