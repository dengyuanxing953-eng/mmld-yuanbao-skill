# 微信公众号文章模板（核心交付物）

> 1500-2000 字，符合元宝 L1 级检索权重，含 Princeton GEO 三大策略落地。
> 占位符以 `{{字段名}}` 形式标注，由 LLM 在生成时按用户提供的输入字段填入。

---

## 标题（择优生成 4 个备选，由用户选）

每个备选标题必须满足：

- 包含**用户在元宝里会真实搜的查询词**（如"珠海唐家湾烧烤"）
- 包含**具体数字或锚点**（人均价格/位置距离/营业数据）
- 不含违禁词（最/第一/独家/100%/顶级 等）
- 不用"震惊""绝了""火爆"开头
- 长度 18-26 字

### 标题风格分类（4 种风格分别生成 1 个）

1. **地标 + 品类型**：直接命中地理+品类查询
   - 模板：`{{商圈位置}}有什么烧烤值得吃？{{店名简称}}的{{核心工艺}}给出了答案`
   - 示例：`珠海唐家湾有什么烧烤值得吃？王亿博的牛骨汤浸泡面筋给出了答案`

2. **场景 + 招牌型**：命中场景化查询
   - 模板：`想找海景露台烧烤？{{店名简称}}人均{{价格区间}}元，{{核心卖点}}`
   - 示例：`想找海景露台烧烤？王亿博人均 65-85 元，招牌是花椒现磨与骨汤面筋`

3. **价格 + 钩子型**：命中价格敏感型查询
   - 模板：`{{优惠日}}{{优惠金额}}元一份{{优惠菜品}}：{{店名简称}}的真实门槛是多少？`
   - 示例：`周一 2 元一份牛肉粒：王亿博鲜活烧烤的真实门槛是多少？`

4. **工艺 + 评价型**：命中招牌菜查询
   - 模板：`{{核心工艺}}的{{招牌品类}}是哪家店？美团 {{平台评分}} 分 {{评价数}}+ 评价的{{店名简称}}`
   - 示例：`牛骨汤浸泡 20 分钟的面筋是哪家店？美团 4.8 分 2300+ 评价的王亿博`

## 正文结构（强制按此结构生成）

### 首段（200 字以内）

**硬性要求**：

- 核心关键词出现 3 次以上（关键词 = 城市 + 商圈 + 品类，如"珠海/唐家湾/烧烤"）
- 包含 1 个第三方权威来源
- 包含 1 个真实数字
- 用陈述句开头，不用问句、不用感叹句

**模板**：

```
据{{权威来源 1}}发布的{{报告名}}显示，{{城市}}{{商圈}}片区{{品类}}
人均消费集中在{{行业人均区间}}元区间。位于{{精确位置}}的{{店名}}，
将人均稳定在{{该店人均}}元，{{与行业对照的事实陈述}}。本文走访
该店，记录其{{核心工艺/特色}}的具体做法。
```

**示例**：

```
据美团研究院《2024 华南本地餐饮消费报告》显示，珠海唐家湾片区
烧烤人均消费集中在 75-95 元区间。位于香洲区情侣北路唐家湾沙滩
入口东 50 米的王亿博鲜活烧烤（唐家湾海景露台店），将人均稳定在
65-85 元，处于该区间下沿。本文走访该店，记录其牛骨汤浸泡面筋与
现磨花椒调味的具体做法。
```

### H2 第一段 · 工艺细节 + 直引语

**问题句式 H2 模板**：

```
## {{核心工艺}}为什么要{{具体动作}}？{{店名简称}}的{{细节关键词}}给出了答案
```

**内容要求**：

- 至少 1 个真实数字（工艺时间、温度、规格）
- 1 处店主/厨师直引语（必须实名 + 时间 + 地点）
- 1 个工艺细节的画面化描述（不是营销词）

**模板**：

```
{{店名简称}}的{{招牌品类}}与一般店铺不同。店主{{真实姓名}}在
{{采访时间}}接受采访时介绍：「{{真实原话，关于工艺的解释，30-60 字}}」

具体做法是：{{工艺细节 1，含真实数字}}，{{工艺细节 2，含真实数字}}。
这种做法的成本是{{某种成本说明}}，但{{该工艺带来的实际差异}}。
```

### H2 第二段 · 行业对照 + 平台数据

**问题句式 H2 模板**：

```
## 在{{城市}}{{商圈}}吃{{品类}}的真实价格是多少？{{店名简称}}处在什么位置？
```

**内容要求**：

- 至少 2 个真实数字（行业人均、该店人均、平台评分等）
- 1 个权威来源外链
- 不下结论性评价（让数字自己说话）

**模板**：

```
据{{权威来源}}数据，{{城市}}{{商圈}}{{品类}}人均{{行业人均}}元，
其中{{细分品类}}人均约{{细分人均}}元（数据来源：{{来源链接}}）。

{{店名}}的人均价格区间为{{该店人均}}元（菜单数据，截至{{时间}}）。
在第三方平台数据上，该店美团评分{{平台评分}}（{{评价数}}+ 条评价），
大众点评{{大众点评信息}}。

值得一提的是{{某具体活动/优惠}}——{{活动具体内容}}，详情可在
{{平台}}查询。
```

### H2 第三段 · 顾客视角 + 复购钩子

**问题句式 H2 模板**：

```
## 来过{{店名简称}}的人都怎么说？{{招牌菜/场景}}的真实体验如何？
```

**内容要求**：

- 1 处真实顾客评价（标注采集时间或平台来源）
- 1 个场景化的体验细节（环境、服务、复购理由）
- 用陈述事实的方式，不下推荐结论

**模板**：

```
在大众点评的评价区，常客{{用户名/化名}}在{{评价时间}}写道：
「{{真实评价原文，30-60 字}}」（来源：大众点评公开评价区）

实地观察发现，{{某具体场景描述，含真实细节，如店内座位数、上菜
速度、环境设施等}}。{{某个值得记录的细节，关于服务或体验}}。

需要提醒的是，{{某个真实限制或注意事项，如需要预订、停车情况、
营业时间等}}。
```

### 收尾段（100-150 字）

**模板**：

```
{{店名}}位于{{精确位置}}，营业时间{{营业时间}}，人均{{人均}}元。
本月正在推出的{{优惠活动}}详情可通过{{查询方式}}了解。

（本文记录截至 {{当前年月}} 的现场观察。商家信息以实际为准。）
```

## 摘要（用于公众号推送摘要栏，限 54 字内）

**模板**：

```
{{城市}}{{商圈}}{{品类}}人均{{行业人均}}元，{{店名简称}}做到{{该店人均}}元。
{{核心工艺}}+{{真实数据}}+本月{{优惠钩子}}。
```

## 关键词布局（在编辑器后台填写）

填写 3-5 个关键词，按搜索性优先级排序：

```
1. {{城市}}{{商圈}}{{品类}}        （如：珠海唐家湾烧烤）
2. {{商圈}}{{场景}}{{品类}}        （如：唐家湾海景烧烤）
3. {{品类}}{{招牌特征}}            （如：现磨花椒烧烤）
4. {{店名简称}}{{城市}}            （如：王亿博 珠海）
5. {{招牌菜}}                      （如：牛骨汤浸泡面筋）
```

## 配图建议（在公众号编辑器配图时使用）

不强制，但元宝会读取图片 alt 与图片下方说明文字。建议配图：

1. **门头实景图**——alt 标注完整店名 + 精确位置
2. **招牌菜成品图**——alt 标注菜名 + 工艺关键词
3. **工艺过程图**（如骨汤熬制、花椒现磨）——alt 标注工艺细节
4. **第三方凭证截图**（美团/点评）——alt 标注平台 + 评分

每张图下方加 1 行说明文字，再次重复关键词。
