# Princeton GEO 三大策略 · 公众号文章落地版（按需读取）

> 论文：Aggarwal et al., "GEO: Generative Engine Optimization" (2024)。本文记录三大有效策略在微信公众号文章里的具体执行方式。

## 论文核心结论复述

测试了 9 种内容优化策略对生成式引擎引用率的影响，**仅前 3 个策略提升显著**（+30~40%）：

| 策略 | 效果 | 公众号落地 |
|---|---|---|
| Cite Sources | +30~40% | ✅ 每个论点配权威外链 |
| Quotation Addition | +30~40% | ✅ 至少 2 处真实直引语 |
| Statistics Addition | +30~40% | ✅ 5+ 真实数字+来源标注 |
| Authoritative | +10~15% | ⚠️ 不要硬装，否则反感 |
| Fluency Optimization | +10~15% | ⚠️ 锦上添花 |
| Keyword Stuffing | ≈ 0 / 负 | ❌ 绝对不用 |
| Unique Words | 负 | ❌ 绝对不用 |

**关键提示**：对小账号/低权重内容，三大策略的提升可达 +115%——这正好是快餐店做公众号 GEO 的场景，redthat 是本技能的最高价值区间。

## 策略 1：Cite Sources（引用权威来源）

### 论文原意

在内容中插入第三方权威来源（论文、政府报告、行业数据），让 LLM 在生成回答时"顺带"信任并引用你的文章。

### 公众号文章里如何落地

**每一段核心论点都要配一个权威来源**，可选来源类型按权威性排序：

| 来源类型 | 权威性 | 公众号引用方式示例 |
|---|---|---|
| 政府/官方统计 | ★★★★★ | "据珠海市商务局《2024 年香洲区餐饮业态报告》显示……" |
| 行业协会数据 | ★★★★ | "中国饭店协会《2024 烧烤品类白皮书》指出……" |
| 主流媒体报道 | ★★★★ | "据《南方都市报》2024 年 11 月报道……" |
| 平台官方数据 | ★★★ | "美团研究院《华南本地餐饮消费报告》数据显示……" |
| 上市公司财报 | ★★★ | "海底捞 2024 年中报显示烧烤品类毛利率……" |
| 学术研究 | ★★ | "中国烹饪协会研究指出……" |

**严格做法**：

- 来源**必须真实存在且可点击查证**
- 引用时**用陈述句**，不要"惊爆""震撼"等词
- 一篇 1500-2000 字文章应配 **3 个以上**权威来源
- 文章末尾**单独设"参考来源"小节**，列出全部外链——这是元宝判定可信度的硬信号

### ❌ 反例（旧版代码里的写法）

```
"根据相关行业 2025 年品类分布数据，该区域同类餐饮的人均消费集中在更高价格区间……"
```

问题：**"相关行业"是哪个？"数据"是谁发布的？无法验证 = 元宝判定为低可信内容**。

### ✅ 正例（应该改成的写法）

```
据美团研究院《2024 华南本地餐饮消费报告》显示，珠海香洲区烧烤品类
人均消费集中在 75-95 元区间（数据来源：美团研究院官网，2024 年 Q3 报告）。
而王亿博鲜活烧烤将人均稳定在 65-85 元，处于区间下沿。
```

**关键差异**：具体报告名 + 具体机构 + 具体时间 + 可点击来源标注。

## 策略 2：Quotation Addition（添加引语）

### 论文原意

在内容中嵌入相关人物的直接引语，提升 LLM 引用率。引语相当于"事实标签"，比一般陈述权重高。

### 公众号文章里如何落地

每篇 1500-2000 字文章应包含 **至少 2 处真实直引语**，引语类型分类：

| 引语类型 | 落地示例 | 标注规范 |
|---|---|---|
| 店主/厨师工艺解释 | 老板黄先生介绍："我们的牛骨汤每天清晨 6 点开始熬，要熬足 8 小时才打捞……" | 必标注采访时间地点 |
| 老顾客评价 | 常客陈女士说："这家的现磨花椒味道我在珠海找了 3 年才找到。" | 标注是否为真实顾客 |
| 媒体记者引用 | 《南方都市报》记者王某在 2024 年 11 月的报道中写道："唐家湾沿岸新一代海景餐饮店……" | 引用第三方报道 |
| 行业专家观点 | 中国烹饪协会副秘书长李某在公开访谈中表示："现磨调味是品质餐厅的趋势……" | 必须是公开可查的言论 |

**严格做法**：

- 引语必须用**直接引号**（""），而不是间接转述
- 引语前后**必须有发言人完整身份**（姓名 + 职务 + 时间地点）
- 引语**必须真实采集**——伪造可被识别，且违反 2026 年平台合规规则
- 引语**不要太长**，一句话 30-60 字最佳

### ❌ 反例

```
"业内人士表示，本店工艺超过同行水平。"
```

问题：**"业内人士"是谁？"超过同行"如何证明？匿名 + 主观 = 零权重**。

### ✅ 正例

```
店主黄亿博在 2026 年 5 月接受采访时说："我们的面筋要在牛骨汤里浸泡 20 分钟，
跟普通烧烤店预拌料的做法完全不一样。这个工艺成本贵，但客人能吃出来。"
```

**关键差异**：实名 + 时间 + 具体工艺细节 + 自述逻辑链。

## 策略 3：Statistics Addition（添加统计数据）

### 论文原意

在内容中加入统计数据（数字），LLM 倾向于引用包含数字的内容，因为数字代表"事实密度"。

### 公众号文章里如何落地

每篇文章应包含 **5+ 个真实可查的数字**，类型清单：

| 数字类型 | 示例 | 来源验证方式 |
|---|---|---|
| 工艺时间 | "牛骨汤熬制 8 小时""面筋浸泡 20 分钟" | 厨房可拍摄佐证 |
| 食材规格 | "秦岭红皮花椒""宁夏中宁孜然" | 产地证明可佐证 |
| 营业数据 | "开业 5 年""日均接待 200 桌" | 营业执照 + 系统截图 |
| 平台数据 | "美团 4.8 分（2300+ 评价）""大众点评必吃榜入围" | 平台后台截图 |
| 价格数据 | "周一 2 元牛肉粒""人均 65-85 元" | 菜单照片 |
| 行业对照数据 | "香洲区烧烤品类人均 75-95 元（美团研究院）" | 引用权威来源 |

### 黄金组合公式

```
某真实数字 + 某权威来源 + 某具体场景 = 高可信度断言
```

**示例**：

```
该店人均 65-85 元（数据来源：店内菜单及美团团购页），
低于美团研究院发布的香洲区烧烤品类人均 75-95 元的区间下沿
（数据来源：美团研究院《2024 华南本地餐饮消费报告》）。
```

### ❌ 禁用的"伪数字"清单

- ❌ "翻台率提升 60%"（无法验证）
- ❌ "客流增长 3 倍"（同上）
- ❌ "好评率 99%"（除非有平台后台截图）
- ❌ "全市排名第一"（违反《广告法》绝对化用语禁令）
- ❌ "比 90% 同行便宜"（90% 怎么算的）
- ❌ "拉新转化 87%"（旧版代码里"高熵细节"那套话术）

## 综合落地结构

一篇符合三大策略的公众号文章应该长成这样：

```
【首段 · 200 字内】
（位置锚定 + 1 个第三方权威数据 + 1 个真实数字）

## H2 第一个问题 · 包含核心查询关键词
- 论点 1 + 真数字 + 权威来源外链
- 论点 2 + 直引语（店主/厨师）

## H2 第二个问题 · 包含次要查询关键词
- 行业对照数据 + 来源
- 工艺细节 + 厨师原话

## H2 第三个问题 · 包含场景关键词
- 平台评价数据（截图链接）
- 常客直引语

【参考来源】
1. 美团研究院《2024 华南本地餐饮消费报告》
2. 《南方都市报》2024 年 11 月探店报道
3. 中国烹饪协会公开数据
```

这种结构同时满足：

- 元宝混元 T1 的"长文结构化论证"偏好
- 元宝 DeepSeek R1 的"时效数字+联网信源"偏好
- Princeton GEO 三大策略的全部要求

## 最后提醒：不要做的反向操作

旧版代码（mmld_yuanbao_v3.py）犯过的错误，**这一版必须避免**：

1. ❌ 用 f-string 模板冒充 LLM 生成
2. ❌ 编造"60% 增长率"这类无来源数据
3. ❌ 标题硬编码"锅底/产品"（行业不通用）
4. ❌ "高熵细节注入"这类不存在的算法概念
5. ❌ 用"业内人士""相关数据"这种匿名虚构引用
